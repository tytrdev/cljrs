Platformer — live coded

A 2D platformer whose entire logic lives in the editor below. Physics (kinematic AABB), gameplay constants, and level geometry are all Clojure data. Edit anything — JUMP-V, GRAVITY, platforms, the step fn itself — and it takes effect immediately without resetting the player's position, because the state atom is declared with defonce. Click the stage to focus, then control with the keys below.

←/A + →/D move · ↑/W/Space jump · R respawn

—

What's happening

  • (defonce state ...) creates the world state atom exactly once. Re-evaluating the source doesn't touch it.
  • (def JUMP-V -13) ... are ordinary defs — editing them redefines the constants live. Move the player off a platform, then change GRAVITY to 0.2: the next frame's fall is gentler.
  • (defn step [w L? R? J?] ...) is pure: takes state + input, returns new state. Swapping the body changes behavior on the very next tick.
  • Because state is plain data and step is pure, the door is open for pause / rewind / future-trajectory scrubbing — next phase.
