Library reference

Grouped reference for everything that ships with cljrs — the core Clojure stdlib, the GPU shader stdlib, the ML autograd toolkit, browser-side interop, the hiccup UI layer, the music-theory toolkit, and the Rust-interop story. Each page mixes signature-level documentation with editable, runnable examples that execute in the same wasm Repl that powers REPL.

For a flat coverage matrix (everything implemented, vs. Clojure JVM), see Coverage. This section is meant for browsing and learning by example, more like rustdoc than a feature checklist.

Modules

PageNamespaceWhat it covers
core cljrs.core Arithmetic, collections, sequences, control flow, atoms, regex, math.
shader demo_gpu/stdlib.clj Macro stdlib for defn-gpu-pixel kernels: noise, SDFs, color packing.
ml cljrs.ml Tensor autograd: matmul, relu, mse, backward, sgd-step.
js cljrs.js Browser DOM, events, localStorage, fetch. wasm-only.
ui cljrs.ui Hiccup → HTML, mount/hydrate, prerender story.
music cljrs.music Note parsing, scales, chords, progressions, arpeggios.
rust interop (any) Opaque handles, install pattern, bindgen, adding a new crate.
