Clojure surface-area coverage

This page is a running inventory of what Clojure features cljrs currently supports. "Supported" means there's a test for it. Gaps are explicit — please open an issue if you want something prioritized.

Callable-value polymorphism

Keywords, maps, sets, and vectors are all callable — standard Clojure. 

(:name {:name "ty"})        ; ⇒ "ty"
({:a 1 :b 2} :a)            ; ⇒ 1
(#{1 2 3} 2)                ; ⇒ 2
([10 20 30] 1)              ; ⇒ 20

What this page isn't

This is not a performance matrix — see Benchmarks. It's not a spec — Clojure doesn't have a formal one either. It's a "will my code probably run" reference, and when in doubt the REPL is authoritative.
