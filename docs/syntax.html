Native-compiled fns

The defn-native form JIT-compiles a typed body to machine code via MLIR. Browser WASM bundles don't have a JIT (melior won't cross-compile), so these fall back to the tree-walker on this page — but running cljrs natively, fib here runs at native-Rust speeds. See Benchmarks. 

(defn-native fib ^i64 [^i64 n]
  (if (< n 2) n (+ (fib (- n 1)) (fib (- n 2)))))
