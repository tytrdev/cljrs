defn-native

Native JIT via MLIR. The browser bundle has no JIT, so these fall back to the interpreter here. Running natively, fib hits the benchmarked speeds. 

(defn-native fib ^i64 [^i64 n]
  (if (< n 2) n (+ (fib (- n 1)) (fib (- n 2)))))
