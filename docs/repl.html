Live REPL

Stateful session. defs persist across runs; hit ⌘↵ or click Run. Runs entirely in your browser — the WASM bundle is ~400KB and loads once.

stateful — defs stick · ⌘↵ to run

What's available

The full tree-walker: ~125 builtins, macro system, persistent collections, destructuring, threading macros, try/catch, atoms. See Coverage for the list.

What's not available here: defn-native falls back to the interpreter (no LLVM in the browser), file I/O (slurp/spit), and the graphics kernels. For those, run cljrs natively.

Things to try

(defn factorial [n]
  (reduce * (range 1 (inc n))))
(factorial 20)
;; Fizzbuzz
(defn fizzbuzz [n]
  (cond
    (zero? (mod n 15)) "FizzBuzz"
    (zero? (mod n 3))  "Fizz"
    (zero? (mod n 5))  "Buzz"
    :else n))
(map fizzbuzz (range 1 16))
;; Macro: trace a fn call
(defmacro trace [expr]
  `(let [v ~expr]
     (println "=>" v)
     v))
(trace (+ 1 2 3))
