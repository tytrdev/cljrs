The full tree walker. About 125 builtins, macros, persistent collections, destructuring, threading, try/catch, atoms. See Coverage.

Not available in-browser: defn-native falls back to the interpreter (no LLVM in WASM), file I/O, graphics kernels. Run cljrs natively for those.

(defn factorial [n]
  (reduce * (range 1 (inc n))))
(factorial 20)
(defn fizzbuzz [n]
  (cond
    (zero? (mod n 15)) "FizzBuzz"
    (zero? (mod n 3))  "Fizz"
    (zero? (mod n 5))  "Buzz"
    :else n))
(map fizzbuzz (range 1 16))
(defmacro trace [expr]
  `(let [v ~expr]
     (println "=>" v)
     v))
(trace (+ 1 2 3))
