Roadmap

Shipped

  • Tree walker, reader, macro system.
  • MLIR-JIT path for type-hinted fns.
  • Persistent collections, destructuring, threading and control macros.
  • Atoms, try/catch, about 125 builtins.
  • Live-coded CPU demos with editor, sliders, time scrubbing.
  • Headless recorder (bin/record, bin/gpu-record).
  • WASM bundle and in-browser REPL.
  • GPU kernel DSL via wgpu (Metal / Vulkan / DX12 / WebGPU).
  • GPU benchmarks against numpy and pytorch.

Next

  1. Clojure coverage: defprotocol, deftype, defrecord, real namespaces with :require / :as, lazy-seqs, transducers, auto-gensym, regex literals, ratios.
  2. More GPU demos: 3D raymarching with lighting, SPH fluid, N-body, softbody.
  3. Motion-compensated TAA for zooming kernels.
  4. In-browser defn-gpu-pixel. Compile cljrs to WGSL inside the WASM bundle, hot-swap the WebGPU pipeline from the REPL.
  5. CUDA and cross-platform GPU benchmarks on a rig with an NVIDIA card.

Non-goals

  • JVM interop. No import of Java classes.
  • Bit-for-bit Clojure compatibility. Match semantics where cheap, diverge where JVM baggage isn't worth it.
  • A package manager. load-file + require until someone wants more.
