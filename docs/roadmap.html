Roadmap

Shipped

  • Tree walker, reader, macro system.
  • MLIR-JIT path for type-hinted fns.
  • Persistent collections, destructuring, threading and control macros.
  • Atoms, try/catch, 300+ host builtins.
  • clojure.core 405/662, plus 100% of clojure.string (21/21), clojure.set (12/12), clojure.walk (10/10), clojure.edn (2/2). See Coverage.
  • defprotocol, defrecord, multimethods, lazy seqs, transducers, regex.
  • Real namespaces with :require / :as / :refer.
  • cljrs.test with deftest / is / testing / run-tests.
  • Live-coded CPU demos with editor, sliders, time scrubbing.
  • Live-coded browser demos: platformer (replay-on-resume), Conway's Life (live rule edits), synth (presets + effects), step sequencer (cljrs.ui + randomize), ML training showcase.
  • Headless recorder (bin/record, bin/gpu-record).
  • WASM bundle and in-browser REPL.
  • GPU kernel DSL via wgpu (Metal / Vulkan / DX12 / WebGPU).
  • GPU shader stdlib: palette, noise, SDF helpers.
  • GPU benchmarks against numpy and pytorch; live in-browser matmul bench vs numpy / JAX / PyTorch references.
  • Reader error reporting with source locations.
  • Rust-crate interop via opaque handles (rapier2d/3d for Physics).

Next

  1. Remaining Clojure coverage: deftype, auto-gensym in syntax-quote, regex literals, ratios, lazy-seq chunking.
  2. More GPU demos: SPH fluid, N-body, softbody.
  3. Motion-compensated TAA for zooming kernels.
  4. In-browser defn-gpu-pixel. Compile cljrs to WGSL inside the WASM bundle, hot-swap the WebGPU pipeline from the REPL.
  5. CUDA and cross-platform GPU benchmarks on a rig with an NVIDIA card.

Non-goals

  • JVM interop. No import of Java classes.
  • Bit-for-bit Clojure compatibility. Match semantics where cheap, diverge where JVM baggage isn't worth it.
  • A package manager. load-file + require until someone wants more.
